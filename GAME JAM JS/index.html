Flappy Duck

Comment jouer ?

Bienvenue dans Flappy Duck !

1. Appuie sur ESPACE pour sauter.
2. Évite les tuyaux.
3. Gère ta barre de Mana.
4. La touche échap permet de mettre en pause le jeu
5. Le jeu commence en Gravité Inversée (tu tombes vers le haut !)
6. Appuie sur 0 pour le mode Normal, ou 1 pour le mode Inversé (sur l'écran d'accueil).
7. Fais le meilleur score possible !

Bonne chance !
